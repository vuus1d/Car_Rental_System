# Car Rental System

## Project Description
Car Rental System is a console-based Python application for managing car rentals.  
The program allows users to rent and return cars, view statistics, and save data using JSON files.

---

## Features
- Show available cars
- Rent a car
- Return a car
- Save data to JSON
- Load data from JSON
- Show statistics
- Error handling
- Logging decorator

---

## Technologies Used
- Python
- JSON
- Object-Oriented Programming (OOP)
